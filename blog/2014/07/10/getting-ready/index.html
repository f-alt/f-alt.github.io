getting ready!

2014-07-10

DC22 is only 4 weeks away! As this is my first one, I want to make sure I’m extra prepared. I really don’t want to end up on the wall of sheep!

First step is securing my laptop. I’m taking my trust m11x (R1) since it has like a 6 hour battery life. I haven’t started configuring the software yet since I’m still waiting to get my hands on a second harddrive (which is probably going to get slagged, or trapped in a fish bowl). It’s going to be running some debian or ubuntu derivative. I’m tempted to run my own openvpn server, but I’m pretty sure that’s just asking for trouble, so I’ll be buying vpn access for the length of the con. I will also be using iptables to ensure that no connection gets out until the vpn tunnel has connected.

Second step is to secure my phone. I found ready sim which looks like an excellent for use at defcon. I’m working under the assumption that we’re at the point where people shouldn’t just worry about rogue wi-fi APs, but rogue cell stations too! Thankfully my phone will be rooted, so I can ensure that all my traffic is over a VPN.

Final step is setting up this box. I figure at some point I’ll need to host something (a tracker, a wiki, an irc server), but I’d rather not use a box that’s connected to me right now, and I need to be able to press the destroy button when the con is over.

I’ll post more howtos later with specifics.
